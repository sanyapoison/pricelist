Таблица – совокупность столбцов и строк (записей), в которых упорядочено, хранится используемая информация.


Для просмотра имеющегося списка таблиц, необходимо перейти в раздел «Таблицы»


В списке таблиц можно просмотреть все созданные ранее таблицы, причем количество созданных таблиц и записей в них не ограничено. Для выбранной из списка таблицы возможны посмотр столбцов и записей (строк), созданных ранее или добавить новые.


Для добавления новой записи введите необходимое значение (строка или число) и нажмите кнопку добавить.


Примечание: чтобы у вас была возможность вписывать данные в таблицу, предварительно создайте столбцы в разделе «Управление». Для удаления записей – поставьте галочку напротив записи, нажмите на кнопку удалить ( ) и выберите «удалить выбранное» либо нажмите на меню управления записью ( ) и выберите пункт «Удалить запись» ( ).
Примечание: удаление таблиц и столбцов возможно только в разделе «Управление» без возможности восстановления.


Для правки записи нажмите на меню управления записью ( ) и выберите пункт «Изменить запись» ( ). Произойдет переход в режим редактирования записи, при этом строка заменится на набор полей ввода с введенными данными текущей записи и опциональные кнопки «Сохранить запись после правки» ( ) и «Отменить изменение текущей записи» ( ).


Для навигации по контенту доступны фильтр и постраничная навигация.

Раздел «Управление» служит для добавления/изменения/удаления таблиц, столбцов.


Для добавления новой таблицы, в список уже имеющихся (или создания первой таблицы), необходимо:

  • Перейти в раздел «Управление».
  • В поле «Название таблицы», вписать желаемое имя таблицы и нажать кнопку «Добавить».

Примечание: создать две таблицы с одинаковым именем не получится, удалите старую или переименуйте новую таблицу.


Для добавления нового столба, необходимо в списке таблиц (раздел «Управление»), выбрать таблицу, в которую будет добавлен столбец.


Далее откроется еще несколько блоков, для выбора названия и типа столба. В поле «Название столба» необходимо вписать название нового столба (буквы русского и латинского алфавита, цифры и символы), а в поле «Сортировка» - положение в списке.


После добавления нового столбца, он будет отображен в разделе ниже – «Структура Таблицы»


Тип данных можно легко изменить, для этого необходимо нажать на тип столбца (серая овальная обводка вокруг текста) и выбрать необходимый тип. Не забудьте нажать «Сохранить» для запоминания внесенных изменений. Для просмотра полученного результата, необходимо перейти в раздел «Таблицы», выбрать из списка таблиц - созданную ранее и нажать на нее. Справа будет отображена вся информация о столбцах и строках. В таблице ниже в качестве «Числового столбца» используется количество товара на складе, а в «Строковом столбце» - наименование товаров.


Удаление столбцов:

  • Зайти в раздел «Управление»
  • Выбрать нужную таблицу (Список Таблиц)
  • В окне «Структура Таблицы» , напротив столбца, который нужно удалить

Примечание: при удалении столбца, все внесенные данные будут так же удалены.

Расширение XLSX – расширение с файлами данных, рекомендуется для открытия и редактирования в Microsoft Excel 2007 и выше, совместим с более старыми версиями Microsoft Excel и альтернативными офисными пакетами.


После экcпорта таблицы, с ней можно продолжить работу в оффлайн режиме (без доступа к интернету).


Примечание: тип данных столбца менять не рекомендуется, иначе это скажется на работоспособности таблицы и приведет к ошибкам экспорта (например: если в типе данных столбца «Цена» у вас указаны только числа: 100,200,300 и т.д. – крайне не рекомендуется приписывать еще и валюту (руб, usd), название валюты может быть указано в наименовании столбца).


Экспорт таблицы:

  • Перед началом экспорта необходимо выделить нужные таблицы и столбцы нажав на соответствующие чекбоксы
  • Нажать на кнопку «Экспортировать»


Примечание: Название таблицы экспортируется как название Листа в Excel.

CSV – текстовый формат, предназначенный для представления табличных данных.


Перед началом экспорта необходимо:

  • Выбрать нужную таблицу
  • Выделить нужные столбцы
  • Нажать на кнопку «Экспортировать»


Примечание: Название таблицы экспортируется как название файла.

Расширение PDF – открытый формат электронных документов.


Перед началом экспорта необходимо:

  • Выбрать нужную таблицу
  • Выделить нужные столбцы
  • Нажать на кнопку «Экспортировать»


Примечание: Название таблицы экспортируется как название файла и заголовок таблицы. Наименование столбцов и заголовка выделяется полужирным начертанием.

Для импорта выберите файл формата XLSX.


Если желаете импортировать все данные с файла то нажмите на кнопку «Импортировать».


Существует возможность опционально выбрать что импортировать. Для выбора опций уберите галочку «Импортировать всё».


В выпадающем списке будет доступен список таблиц для импорта. Примечание: Таблицы в выпадающем списке являются листами в выбранном для импорте файле формата XLSX.


Если желаете импортировать только структуру уберите галочку «Импортировать данные», таким образом, в вашу таблицу импортируется только структура.
Если желаете импортировать только данные уберите галочку в поле «Импортировать структуру» таким образом, в вашу таблицу импортируются только данные.

Для импорта выберите файл формата CSV.


Если желаете импортировать все данные с файла то нажмите на кнопку «Импортировать».


Существует возможность опционально выбрать что импортировать. Для выбора опций уберите галочку «Импортировать всё».


Если желаете импортировать только структуру уберите галочку «Импортировать данные», таким образом, в вашу таблицу импортируется только структура.
Если желаете импортировать только данные уберите галочку в поле «Импортировать структуру» таким образом, в вашу таблицу импортируются только данные.
